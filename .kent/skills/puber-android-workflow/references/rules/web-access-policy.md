# Web Access Policy

Fetching known public technical documentation is allowed when relevant:

- Android, Kotlin, Gradle, Compose, Koin, Ktor, OkHttp, Coil, Voyager, Detekt, and GitHub documentation.
- Kent documentation under `https://kent.sh/*`.

Ask before fetching:

- URLs that may contain private project context.
- Issue tracker URLs.
- Private repository URLs.
- Any URL containing credentials, tokens, or local-only identifiers.

Generic public technical searches are allowed only when they do not include private identifiers.
